# C3 Notes Release Notes

Version: `v0.5-scm`

## Summary

This release captures the configuration management milestone for C3 Notes. It focuses on branch workflow discipline, documentation maturity, and a more complete feature set for the authenticated notes platform.

## Included Changes

- Added authenticated multi-user notes support
- Added note archiving, pinning, deadlines, priorities, and checklist fields
- Improved account settings and profile image upload flow
- Added curated UI theme switching
- Improved deployment readiness for Render and Vercel
- Expanded coursework and project documentation

## Known Limitations

- Evidence such as PR screenshots, merge confirmation, and pushed Git tags must still be completed through GitHub workflow activity.
- Persistent production media storage is not yet externalized for uploaded avatars.

## Recommended Next Steps

- Complete SCM evidence through branch, PR, merge, and tag workflow
- Expand automated test coverage
- Finish final deployment validation with live URLs
